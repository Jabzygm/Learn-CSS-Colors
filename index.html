Basic Colors:

Red: rgb (255,0,0)

this color also has a box-shadow property, which gives it shadows on the sides

the first value is for offsetX, which changes it's offset on the X axis,
the second value, offsetY, changes it's offset on the Y axis
you can also change the color of the shadow, in this case I used pink.

Green: rgb (0,127,0)

as you can see, box-shadow on this color is different. it is more blurred, because I added the third value, which is called blurRadius

Blue: rgb (0,0,255)

If you want spread the shadow out further, you can add the third "spreadRadius" value

Yellow: rgb (255,255,0)

255 is the maximum value and 127 is a half in the "rgb" function, you can also use actual color names (e.g: red, blue, yellow etc)

You can also use the "hsl" function, where first value is for hue, second value is for saturation, and the third one is for lightness

to make the color more or less opaque, we can use the "opacity" property
